# TableViewCellHeight-AutoLayout-ManualLayout

在TableViewCell中, 经常使用 UILabel 与 UITextView 来存放cell中的文字, 而计算高度却经常产生问题. 

本Demo实现了:
1.通过AutoLayout 与 ManualLayout 的两种方式来获取高度.
2.对TextView输入内容即时更新Cell高度.
