###说明
---
在TableViewCell中, 经常使用 UILabel 与 UITextView 来存放cell中的文字, 而计算高度却经常产生问题. 本项目通过不同方式解决此问题.


###功能
---
* 通过AutoLayout 与 ManualLayout 的两种方式来获取高度.
* 对TextView输入内容即时更新Cell高度.
* iOS8 EstimatedHeightForRow
